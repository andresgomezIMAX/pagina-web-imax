Gestión y Supervisión de Obras (GSO)

Volver

Gestionamos y acompañamos todo el proceso de ejecución de obras para todo tipo de proyectos. Desde el proceso de Planificación, Contratación, Pre-construcción (Preliminares), Supervisión Permanente y Conformidad. El servicio se acomoda a los requerimientos que se deseen controlar, de forma parcial o total dentro de las diferentes etapas de ejecución del proyecto.

Servicios:

  • Supervisiones Permanentes.
  • Gerencias de Obra.
  • Saneamientos Físicos.
